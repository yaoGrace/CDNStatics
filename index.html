命令行工具

phpGrace 需要命令行形式来实现初始化站点、创建控制器、模型等等自动化工作。所以在开始前请保证您的 php 已被添加至环境变量。

将 php 添加至环境变量

1 windows

1 确认 php 安装文件位置;
2 我的电脑右键  > 属性 > 高级 > 环境变量设置
3 点击 path 添加 php 安装目录/bin 目录到环境变量
4 测试 : 
命令行输入 php -v, 返回 php 版本信息代表环境变量设置成功~

2 macOS

# 以下操作基于命令终端

1 查询系统环境变量文件位置
echo $SHELL
返回 如 : /bin/zsh 可以看出

2 添加 php 到环境变量
vim ~/.zshrc
添加 :
export PATH="/Applications/phpstudy/Extensions/php/php7.3.11/bin:$PATH"
注意 : php 的 bin 目录具体位置按您的实际安装位置填写

3 刷新环境变量使其生效
source ~/.zshrc

4 测试命令
php -v
返回 php 版本信息代表环境变量设置成功 ~

使用终端

您可以通过各类终端工具使用 grace 命令，更推荐您在编辑器内置终端内使用 grace 命令。

phpGrace 命令列表

// 注意在项目根目录下使用下面的命令
创建新站点 : php grace cw 站点名称
重构伪静态 : php grace cht 站点名称
创建新分组 : php grace cg 分组名称 in 站点名称
创建控制器 : php grace cc 控制器名称 站点或分组
创建新模型 : php grace cm 模型名称
创建小部件 : php grace cwt 部件名称
创建演示数据表 : php grace cdt

通过上面的命令，您就可以快速的实现 命令方式 创建分组、控制器、模型文件。
